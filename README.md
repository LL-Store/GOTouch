# [GOTouch](https://github.com/LL-Store/GOTouch)
一款包含浏览器功能的桌面应用

> 支持`Windows`和`MacOS`电脑的使用（如果希望支持更多端，可以[ 留言 ](https://github.com/LL-Store/GOTouch/issues)告诉我们）！

## 如何开发？

确定node.js已经安装好了以后，直接执行下面目录安装依赖：

```js
npm install
```

然后，执行下面的命令即可启动开发：

```js
npm run dev
```

开发完毕，需要打包发布，直接：

```js
npm run build
```

打包后的安装包位于 release 中。
